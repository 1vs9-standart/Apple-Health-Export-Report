# Отчёт по данным Apple «Здоровье»

## 1. Скачайте данные с iPhone

1. Откройте **Здоровье** → вкладка **Профиль** (иконка человека).
2. Пролистайте вниз → **Экспорт данных о здоровье**.
3. Дождитесь архива и сохраните ZIP на компьютер.

## 2. Положите файлы в папку `data`

Распакуйте ZIP. Внутри найдите и скопируйте в папку **`data`** этой программы:

- `export.xml` — обязательно
- папку `electrocardiograms` — если есть (ЭКГ с часов)
- папку `workout-routes` — если есть (маршруты тренировок)

Файл `export_cda.xml` — по желанию, если он есть в архиве.

## 3. Установите Python (один раз)

Если Python ещё не установлен: скачайте с [python.org](https://www.python.org/downloads/) и при установке отметьте **Add Python to PATH**.

## 4. Запустите отчёт

**Проще всего (Windows):** дважды щёлкните **`run.bat`** в этой папке. Отчёт откроется в браузере сам.

Или вручную в терминале:

```bash
pip install -r requirements.txt
python health_insights.py
```

## 5. Откройте результат

Готовый файл: **`output/health_report.html`**

---

**Не работает?** Проверьте, что `export.xml` лежит в папке `data`, а не в корне проекта.

**Другой период** (например, полгода вместо года):

```bash
python health_insights.py --days 365
```

Это не медицинский диагноз — только наглядная сводка по вашим данным.
